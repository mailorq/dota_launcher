# Voice Dota 2 launcher

Слушает микрофон и запускает Dota 2 когда вы произносите заветную фразу.

Создатель: mailor

## Как работает

1. Слушает микрофон
2. Распознаёт речь с помощью воск
3. Преобразует голос в текст
4. Проверяет наличие фразы
5. Запускает Доту

## Требования

- Python 3.8+
- Микрофон

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```
